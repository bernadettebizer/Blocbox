Case Study

Blocbox


A website for editing and storing notes, images, and links

Overview

Blockbox is a SaaS (Software as a Service) web application which allows people to collect information and store it in a single place. A person can create simple documents, store images, links to websites, and collaborate with others.

Challenge

The main challenge for presenting an information storing and organizing SaaS is making the materials simple to access and edit.

This simplicity needed to be evident in the following workflows:
  • Creating an account
  • Adding content
  • Editing content
  • Organizing content

Solution

The focus for this project was to craft an intuitive and simple platform to organize and access stored materials. A consistent method of adding and accessing materials makes it so that users can focus on their own created content instead of thinking about how to accomplish an action. Keeping this in mind, I made it a priority to design a solution that had a consistent design that was simple to navigate.
